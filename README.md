===========================
PROGRAMMING ROADMAP (6+ MONTHS)
===========================

GOAL:
Become good at programming, then move into C++, computer internals, and advanced software engineering concepts.

--------------------------------
SETUP (Day 1)
--------------------------------

Install:
- Python
- Visual Studio Code
- Git
- GitHub account

Create folders:
Programming/
    Python/
    Cpp/
    Projects/

--------------------------------
PHASE 1 - PYTHON (Weeks 1-4)
--------------------------------

Learn:
- Variables
- Data Types
- Input/Output
- If Statements
- Loops
- Functions
- Lists
- Dictionaries
- Modules
- Files

Learn from:
- Corey Schafer (YouTube)
- Programming with Mosh (YouTube)

Practice after EVERY lesson.

Projects:
- Calculator
- BMI Calculator
- Number Guessing Game
- Rock Paper Scissors
- Password Generator
- Quiz Game
- To-Do List

Run your code:
VS Code → Terminal

python filename.py

--------------------------------
PHASE 2 - GIT (Week 5)
--------------------------------

Learn:
- git init
- git add
- git commit
- git push

Upload every project to GitHub.

--------------------------------
PHASE 3 - BIGGER PYTHON PROJECTS
(Weeks 6-8)
--------------------------------

Build:
- Flashcard App
- Expense Tracker
- File Organizer
- Weather App (API)
- Simple Password Manager

Goal:
Learn to solve problems without following tutorials.

--------------------------------
PHASE 4 - C++ (Months 3-4)
--------------------------------

Learn from:
learncpp.com

Install:
Visual Studio Community

Learn:
- Variables
- Functions
- Classes
- Objects
- Pointers
- References
- Memory
- Arrays
- Vectors

Projects:
- Calculator
- Tic Tac Toe
- Inventory System
- Banking System

--------------------------------
PHASE 5 - COMPUTER SCIENCE
(Months 4-5)
--------------------------------

Study:
- Algorithms
- Data Structures
- Memory
- Processes
- Threads
- Networking

Watch:
- CS50
- Low Level Learning
- LiveOverflow

--------------------------------
PHASE 6 - ADVANCED
(Month 6+)
--------------------------------

Learn:
- Windows Internals
- Assembly Basics
- Reverse Engineering Concepts
- Debugging
- Reading Large Codebases

--------------------------------
DAILY SCHEDULE
--------------------------------

20 min → Watch tutorial

40 min → Recreate without looking

40 min → Build your own project

20 min → Debug + upload to GitHub

--------------------------------
RULES
--------------------------------

✓ Code every day.
✓ Build after every lesson.
✓ Google when stuck.
✓ Never just copy tutorials.
✓ Finish projects before starting new ones.
✓ Keep all projects on GitHub.
✓ Focus on understanding, not memorizing.
